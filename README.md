# EngWords

## Телеграм Бот для изучения английских слов

Базово предлагается два режима работы бота:

1. Изучение новых слов
2. Повторение изученных слова
3. Вывод статистики пользователя

* Словарь представлен Oxford Dictionary 3000

* При изучении новых слов, БОТ отправляет пользователю слова на английском, пока не наберется список из 10 новых слов для изучения

* Доступны четыре режима повторения слов - RUS-ENG, ENG-RUS, ввод ответа кнопками или текстом

* Повторение слов происходит по следующему принципу: бот по очереди присылает по 10 слов, 7 слов из изучаемых, 3 слова из уже известных. Слова из изучаемых отправляются по принципу слов с наименьшим количеством повторений

Использованы:
- Python 3.11
- pyTelegramBotApi
- SQLAlchemy
- DB: Postgres
- asyncio
