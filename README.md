# EngWords

## Телеграм Бот для изучения английских слов

Базово предлагается два режима работы бота:

1. Изучение новых слов
2. Повторение изученных слова
3. Вывод статистики пользователя

* При изучении новых слов, БОТ отправляет пользователю 5 новых английских слов с переводом.
После этого, новые слова добавляются в изученные пользователем

* При повторении изученных слов, БОТ отправляет изученные слова по одному и предлагает в виде кнопок 4 варианта ответа

Использованы:
- Python 3.11
- pyTelegramBotApi
- SQLAlchemy
- DB: Postgres
